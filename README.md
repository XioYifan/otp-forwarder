# OTP Forwarder Bot

Bot ini menggunakan [Telethon](https://docs.telethon.dev/) untuk mem-forward pesan
dari bot tertentu (`SOURCE_BOT`) ke channel pribadi (`TARGET_CHANNEL`).

## Deploy di Render
- Tambahkan Environment Variables di Render Dashboard:
- API_ID
- API_HASH
- SOURCE_BOT
- TARGET_CHANNEL
